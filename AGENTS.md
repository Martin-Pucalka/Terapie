# AI Agent Instructions for Terapie Website

## Project Overview
This is a static website for Pavla Mudrochová, a therapist in Olomouc, Czech Republic. The site provides information about her services, experience, and contact details.

## Tech Stack
- **Frontend**: HTML5, CSS3, vanilla JavaScript
- **Deployment**: Static hosting (GitHub Pages via CNAME)
- **No build tools**: Direct file editing and upload

## Key Conventions
- **Language**: Content is in Czech (cs), code and comments can be in English
- **Accessibility**: Use semantic HTML with proper ARIA labels and roles
- **Images**: Provide WebP format with JPEG fallbacks using `<picture>` elements
- **Responsive**: Use CSS clamp() and modern layout techniques
- **SEO**: Maintain meta tags, sitemap.xml, and robots.txt

## Development Workflow
- Edit HTML/CSS/JS files directly
- Test locally by opening index.html in browser
- Deploy by committing changes (GitHub Pages auto-deploys)

## Contact Form
Uses formsubmit.co service for AJAX form submission. Handle success/error states in JavaScript.

## File Structure
- `index.html`: Main page with all sections
- `style.css`: Styles with modern CSS features
- `images/`: Optimized images and favicons
- `robots.txt`, `sitemap.xml`: SEO files
- `CNAME`: Domain configuration</content>
